Introduce
This is a tool with can help you manage MongoDB easily.
With this tool You can :          
Create or Drop Database,Collection or Document Element by Gui interface.
Monitor the status of MongoDB realtime.
Run Shell functions without need to remember the commands syntax.
Import Data from Access Database (Windows OS Only)
It will be migrated to Mono for Linux OS as soon as possible

Because the tool use the offical C# driver,.NET framework 4.0 is needed.
Tour:Basic

Startup
As it is the first time you run this application,a language picker form will popup:
If English is your mother tongue,you can push OK directly



After language picker,you need to tell application where is your mongo executable path.


In this tour,the mongo executable path is C:\runmongo\bin

If you see the form below,the application is successful startup.
Add Connection
The first thing to operate the mongoDB is add a MongoDB connection.
Please make sure that the mongo Server has been startup.
You can start mongoDB by DOS console like this:

cd C:\runmongo\bin
mkdir C:\mongodb\magicdict
mongod --port 28030 --dbpath C:\mongodb\magicdict



More reference:Quickstart Windows
[A Mongo connection is not only a single server,but also a replset,a route or config server in sharding]

Open the Connection Manager form by menu:
Managerment -> Connection Manager:



This form will list up all connections that you has registered in application.
Because of this is the first time you use this application,the list is empty.
Click “Add” to register a new connection.



The simplest connection only need to fill 3 fields
  • Connection(you can name the connection as you like)
  • Host(the ip or host of server,you can fill ‘localhost’ when you do local test)
  • Port(the port of server)


Pick the connection you registered right now, then click "OK".Now you can operate the mongo Server.
You can collapse or expansion the treeview by MenuItem:
  • Management -> collapse
  • Management -> expansion
Disconnect
If the connection is not need any more,you can disconnect it by:
  • Menu: Opertion -> Server -> Disconnect
  • Contextmenu: Right click the connection treenode to show contextmenu,click the disconnect menuitem

Server properties
This function will show the propery of selected server.
  • Menu: Opertion -> Server -> Server properties
  • Contextmenu: Right click the connection treenode to show contextmenu,click the Server properties menuitem


Some properties will not be shown at some type of connection.

Status
This function can help you monitor the status of server.
  • Menu: Management -> Status




Cluster is useful at a replset.
Server Status need manual refresh

Shut Down
This function will shut down the server.
  • Menu: Management -> Shut Down
  • Contextmenu: Right click the connection treenode to show contextmenu,click the Shut Down menuitem
This function will completely shut down the server!
